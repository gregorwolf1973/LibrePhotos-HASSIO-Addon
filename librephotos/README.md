# LibrePhotos Home Assistant Addon

Selbst gehostete Fotoverwaltung mit KI-Gesichtserkennung, automatischer Klassifizierung und Timeline-Ansicht – direkt in Home Assistant integriert.

## Features

- Automatische Gesichtserkennung und -gruppierung
- KI-basierte Szenen- und Objekterkennung
- Timeline-Ansicht nach Datum
- Albumverwaltung
- Kartenansicht für Fotos mit GPS-Daten
- Unterstützung für RAW-Formate, HEIC und Videos
- Volltextsuche

## Installation

1. Füge dieses Repository zu deinen Home Assistant Addon-Repositories hinzu:  
   `https://github.com/gregorwolf1973/LibrePhotos-HASSIO-Addon`
2. Installiere das "LibrePhotos" Addon
3. Konfiguriere den Fotoordner und das Datenbankpasswort
4. Starte das Addon
