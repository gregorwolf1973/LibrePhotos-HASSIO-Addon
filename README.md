# LibrePhotos HASSIO Addon Repository

Dieses Repository enthält Home Assistant Addons für LibrePhotos.

## Addons

- **LibrePhotos** – Selbst gehostete Fotoverwaltung mit KI-Gesichtserkennung

## Installation

1. Öffne Home Assistant → Einstellungen → Addons → Add-on Store
2. Klicke auf die drei Punkte (oben rechts) → Repositories
3. Füge folgende URL hinzu:
   ```
   https://github.com/gregorwolf1973/LibrePhotos-HASSIO-Addon
   ```
4. Suche nach "LibrePhotos" im Addon Store und installiere es

## Systemanforderungen

- Home Assistant OS oder Supervised
- Mindestens 4 GB RAM
- Mindestens 10 GB freier Speicher (+ Fotobibliothek)
