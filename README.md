# LibrePhotos – Home Assistant Add-on Repository

[!["Buy Me A Coffee"](https://www.buymeacoffee.com/assets/img/custom_images/orange_img.png)](https://buymeacoffee.com/gregorwolf1973)

Run [LibrePhotos](https://github.com/LibrePhotos/librephotos) – a self-hosted, open-source photo management platform with AI-powered face recognition, scene classification and timeline view – as a native Home Assistant add-on.

## Add-ons

### [LibrePhotos](./librephotos)

Self-hosted photo management with AI face recognition, automatic scene/object tagging, map view, RAW/HEIC/video support and full-text search. PostgreSQL, Redis and the LibrePhotos backend + frontend are bundled in a single supervised container.

## Installation

[![Add to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fgregorwolf1973%2FLibrePhotos-HASSIO-Addon)

Click the button → the repository is added to Home Assistant automatically → install **LibrePhotos** from the Add-on Store → Start.

Or manually:

1. In Home Assistant: **Settings → Add-ons → Add-on Store**
2. Top right **⋮ → Repositories**
3. Enter this URL:
   ```
   https://github.com/gregorwolf1973/LibrePhotos-HASSIO-Addon
   ```
4. **LibrePhotos** appears in the store → Install → Start

See [librephotos/README.md](librephotos/README.md) for the feature list and [librephotos/DOCS.md](librephotos/DOCS.md) for the in-depth documentation.

## System Requirements

- Home Assistant OS or Supervised
- Architecture: `amd64` or `aarch64`
- At least 4 GB RAM (8 GB recommended)
- 10 GB free disk space + size of your photo library
